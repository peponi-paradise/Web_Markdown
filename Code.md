## Introduction

<br>

- 단어 또는 구를 `` ` (Backtick) `` 기호를 이용하여 code로 지정

<br>

## Example

<br>

```md
- 글 내에서 `code` 표시
```
```md
- Code 내에 ` (Backtick) 기호가 있는 경우 Backtick 2개로 감싸준다
    `` `code` ``
```
```md
- Code block으로 표현하려는 경우 Backtick 3개로 감싸 작성
- 언어 지정 가능
    ```python
    def TestFunction(PrintString:str):
        print(PrintString)
    ```
```

<br>
