<h3 id="title">Definition Lists</h3>

1. 몇몇 마크다운은 리스트식 정의를 지원한다. 버전 이력 관리, 용어 정의 등에 좋아보인다.
2. 한 줄에 원하는 글을 적고,
3. 다음 줄에 콜론+스페이스 `: ` 기호에 이어 글을 적어준다.
<br>

```md
의자
: 내가 앉아있는 것
: 옷을 걸어두는 용도

키보드
: 매일매일 붙잡고 있는 것
: 20년도 더 쓴 것 같은데 아직도 오른손은 독수리 타자..
: 원한다면 **글자***Formatting*도 [`가능`](#title)
```
