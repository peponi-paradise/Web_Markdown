## Introduction

<br>

- 글에 링크를 다는 기능
- `[링크 이름](링크 주소)` 형식으로 작성

<br>

## Example

```md
- 기본 사용법
    - [티스토리](http://tistory.com)
- 툴팁과 함께 사용
    - [네이버](http://naver.com "네이버")
- 다이렉트 링크 (이메일 또는 URL)
    - <tistory@tistory.com>
    - <https://microsoft.com>
- 링크 강조
   - 강조 링크 **[Bold link](http://tistory.com)**
   - 다른 강조 *[Italic link](https://markdownguide.org)*
- 페이지 내부 링크 연결
   - 여기를 누르면 [Introduction](#introduction)로 이동
- 참조 링크
   - 글 중간에 [참조링크][1] 삽입

<!-- 주소 삽입. 페이지에서는 안보임 -->
[1]: http://tistory.com "Tistory"
```

<br>
